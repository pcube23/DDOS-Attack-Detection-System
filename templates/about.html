About DDoSGuard

🛡️ What is a DDoS Attack?

Distributed Denial of Service (DDoS) attacks overwhelm targets with massive traffic from multiple sources, rendering services unavailable to legitimate users. These attacks can cause significant downtime, financial losses, and damage to reputation.

Common attack types include:

  • SYN Floods: Overwhelming servers with TCP connection requests
  • UDP Floods: Saturating bandwidth with UDP packets
  • ICMP Echo Storms: Flooding networks with ping requests
  • Amplification Attacks: Exploiting protocols to amplify attack traffic

🔍 How We Detect It

DDoSGuard uses advanced machine learning models combined with heuristic analysis to identify attack patterns in real-time network traffic.

Detection Methods:

  • Protocol-specific ML models (ICMP, TCP, UDP)
  • Traffic pattern analysis and anomaly detection
  • Source/destination IP frequency analysis
  • Packet flag and behavior pattern recognition
  • Real-time alert generation with severity classification

📋 Supported File Formats

Upload and analyze network traffic data from various sources:

  • CSV Files: Packet data exported from network analysis tools
  • PCAP Files: Standard packet capture format (.pcap)
  • PCAPNG Files: Next-generation packet capture format (.pcapng)

The system automatically detects file format and extracts relevant packet information including source/destination IPs, protocols, ports, flags, and timestamps.

🔒 Privacy & Security

DDoSGuard processes network packet data locally in your browser and on the server. All analysis is performed on uploaded files without transmitting data to third-party services. Session-based storage ensures your data remains private and secure.

Data Handling:

  • Files are processed server-side and stored temporarily in session cache
  • No persistent storage of sensitive network data
  • All analysis results are session-scoped and cleared on logout
  • Generated reports contain only the data you choose to export
